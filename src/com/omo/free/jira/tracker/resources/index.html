JIRA Shop Tracker UI
Usage Instructions

JIRA Shop Tracker UI was designed for downloading, tracking, and emailing JIRA issues to the ISU shop. This application if used will greatly simplify and automate the downloading and emailing of the JIRA issues to the ISU shop. Presented within the next slides are some explanations and usages of the features that the JIRA Shop Tracker UI provides. Please click the arrows within this window to view the different slides of information.

JIRA Filter Options

(1) Select Project Select the JIRA project to search for issues for.

(2) Filter By Status Select a status to filter your search of JIRA issues. On start up of the app, the issue listing will contain 75 issues of differing statuses.

(3) Search Box Type the number of the issue to run a search for and then click search button.

(4) Refresh Refreshes the JIRA listings by making a connection to JIRA to retrieve the listing of issues for the current project and status selected.

Issues Tab

(1) Select multiple issues from the Open issues listing that you would like to send to the shop via email.

(2) Add and Remove buttons used to add and remove issues to and from the "Issues to send to ISU shop" list box.

(3) Send button used to download JIRA issue(s) and attachments and send them to ISU shop via email.

Sent Issues Tab

(1) View All Tracked Issues used to retrieve a listing of all issues that have been tracked by the JIRAShopTracker.
(2) Select multiple issues from the Tracked issues listing that you would like to send to the shop via email.
(3) Add and Remove buttons used to add and remove issues to and from the "Issues to send to ISU shop" list box.
(4) Send button used to download JIRA issue(s) and attachments and send them to ISU shop.

Menu Bar Options

(1) Report Bug or Enhancement Select this to open a window which will allow you to send a request to fix a bug or an enhancement, via email, to be added to the JIRAShopTrackerUI application.

(2) Connection Settings Select this to open a window which will allow you to change your user id and password used for connecting to the JIRA website.

Report Bug or Enhancement Window

Allows you the ability to request that a bug be fixed or an enhancement be made to the JIRAShopTrackerUI application. During your interaction with this application you may encounter an error or an enhancement idea.

(1) Type Select either BUG or ENHANCEMENT.

(2) Description Enter the description to the bug or enhancement.

(3) Send Press to send the bug/enhancement to the shop.

JIRA User Connection Settings

Allows you to change your JIRA user id and password. It also allows you to save your password for future connections to JIRA.

(1) Enter credentials and press OK button to connect. Password will be saved if you select the Save password checkbox.
